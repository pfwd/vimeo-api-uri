# vimeo-api-uri
A PHP Library that holds Vimeo API URI's. This is used to facade the URI's incase they change in the future

## Install
```
$ composer require pfwd/vimeo-api-uri
```

## Test
```
$ bin/phpstan analyse --level=7 src
```

```
$ bin/codecept run unit
```
